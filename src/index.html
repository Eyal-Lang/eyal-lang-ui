🦁 EyalLang

The no-nonsense, rule-based DSL that interprets Eyal-style directives into Python.

View on GitHub

🚀 Quick Start

pip install eyal-lang

python -m eyal_lang --line "say hello world"

📖 Syntax Examples

# Output
say hello world          → print("hello world")

# Variables
set name to "Eyal"       → name = "Eyal"
set x to 5               → x = 5
set y to x + 10          → y = x + 10

# Math & Greeting
set a to 10
set b to 5
set sum to a + b
say the sum is sum       → print("the sum is sum")

🎯 Features

  • ✅ Zero dependencies
  • ⚡ Fast, deterministic execution
  • 🛠 Extensible rule-based parser
  • 🧠 Clear error messages
  • 📦 PyPI package & CLI-ready
  • 🔒 MIT licensed, open source

💡 CLI Usage

# Run a file
python -m eyal_lang --file examples/hello.eyal

# Interactive mode
python -m eyal_lang

# Save output
python -m eyal_lang --file input.eyal --output out.py
