🦁 EyalLang

A simple, rule-based DSL that interprets Eyal-style directives to Python.

View on GitHub

🚀 Quick Start

pip install eyal-lang

python -m eyal_lang --line "say hello world"

📖 Syntax Examples

# Output
say hello world          → print("hello world")

# Variables
set name to "Eyal"       → name = "Eyal"
set x to 5               → x = 5
set y to x + 10          → y = x + 10

# Math & Greeting
set a to 10
set b to 5
set sum to a + b
say the sum is sum       → print("the sum is sum")

🎯 Features

  • ✅ No dependencies
  • ⚡ Fast, lightweight interpreter
  • 🛠 Extensible rule-based engine
  • 🐍 CLI & API support
  • 🧠 Great error messages
  • 🔒 MIT licensed & open source

💡 CLI Usage

# Run a file
python -m eyal_lang --file examples/hello.eyal

# Interactive mode
python -m eyal_lang

# Save output
python -m eyal_lang --file input.eyal --output out.py
